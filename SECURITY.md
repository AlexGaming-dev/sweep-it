# Security Policy

## Supported Versions
Currently, the following versions of Sweep-It are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0.0 | ❌ No              |

## Reporting a Vulnerability
If you discover a security vulnerability, please **do not** open a public issue. Instead, send an email to **contact@alexgaming.dev**.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Installation

Run this command to install the tool globally:

````bash
npm install -g @alexgamingdev/sweep-it
````

## Usage
Simply run the following command in any cluttered folder to categorize your files:

````bash
sweep-it
````

## Features
- Automatic Sorting: Groups Images, Documents, Code, and Videos.
- Lightning Fast: Built with Node.js for instant organization.
- Clean Workspace: Keeps your `Downloads` or `Desktop` folder tidy.
